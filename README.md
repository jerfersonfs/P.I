## Proposta
A proposta desse projeto é simular como seria se fossemos fazer um sistema real para um cliente, com foco em aplicação prática e portfólio.   
Ele foi desenvolvido a partir do Projeto Integrador do meu curso técnico, onde tive a oportunidade de trabalhar em um cenário próximo ao mercado real.

## O que esse projeto entrega
 - Sistema desktop direcionado a um pequeno negócio específico

## Aprendizados 

### Conceitos técnicos:
  -  Estrutura de um projeto desktop
  -  Organização de código e responsabilidades
  -  Comunicação entre componentes do sistema
  -  Manipulação de dados no sistema 
 
### Conceitos teoricos:
  - Introdução a conceitos de sistemas comerciais
  - Pensamento orientado a soluções
  - Levantamento de requisitos para o desenvolvimento de um sistema
  
## Observação: 
 Esse projeto representa uma etapa importante do meu processo de aprendizado e evolução como desenvolvedor. Continuo realizando melhorias e implementações com o objetivo de aprofundar meus conhecimentos.
